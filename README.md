# BF.NET

## Description 
A full-featured optimizing Brainfuck .NET compiler.


## Usage
```bash
	bfc.exe [OPTIONS] [<path to source file>]
```

### Options
```bash
/p, /code:<code>      Raw Brainfuck code to compile
/d                    Disable code optimization.
/o, /output:<path>    A path to output file
```
